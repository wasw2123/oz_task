### 4. my py
파이썬의 경우 type을 강제하지 않아 생기는 오류를
my py를 통하 강제함으로 해결할 수 있다.

func(a: int, b: int) -> int:
    return a + b
리턴을 받아 다른 곳으로 넘겨준다고 가정하자.
my py가 없다면 a, b가 str으로 들어가면
리턴되는 값도 str으로 나오기에 해당 함수는 정상작동(?) 하지만
이걸 응용한 다른 함수가 오류가 나게 된다
이런점을 방지해 줄 것이라 예상된다
